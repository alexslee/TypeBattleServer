# TypeBattleServer

## Introduction
This server was developed for the sole purpose of being an companion to [TypeBattle](https://github.com/jimmyhoang/TypeBattle).  This server provides the back-end for all the quotes and poems to be used in
TypeBattle.  New poems/quotes can be added to enhance gameplay without needing to release an entire new build of TypeBattle.
With this server a possibility of cross-platform port is available.
